# 🔐 Password Generator (Python)

A simple and beginner-friendly password generator built using Python.
This program creates random passwords based on user preferences such as length, digits, and special symbols.

---

## 🚀 Features

* Generate random passwords of custom length
* Option to include:

  * Digits (0–9)
  * Special symbols (!@#$%^&* etc.)
* Uses Python lists and `random` module
* Beginner-friendly logic (no advanced libraries)

---

## 🛠️ Technologies Used

* Python
* Built-in `random` module
